# ☁️ Cloud-Souverain MA

> Solution d'infrastructure cloud hybride 100% marocaine pour les PME nationales.

## 🎯 Objectif
Offrir aux PME marocaines une alternative sécurisée et performante 
pour l'hébergement de leurs données critiques, garantissant la 
souveraineté numérique du Royaume.

## 🛠️ Technologies
- **Réseau** : Cisco 2911 Router + 3560-24PS Multilayer Switch
- **Virtualisation** : VMware ESXi Hypervisor
- **OS** : Rocky Linux
- **Outil** : Cisco Packet Tracer

## 🏗️ Architecture
- VLAN 20 — 192.168.20.0/24 (Infrastructure interne)
- Accès PME — 10.10.10.0/24 (Réseau client)
- Redundant Storage + Backup Server
- Interface d'administration web sécurisée
